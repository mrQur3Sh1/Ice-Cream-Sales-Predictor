# 🍦 Ice Cream Sales Predictor: Temperature & Weather Analysis  

📊 **Unveiling the relationship between temperature, rain, and ice cream sales using Linear Regression.**  

---

## 🚀 Project Overview  
Ice cream sales fluctuate based on external factors like temperature and rain. This project applies **Linear Regression** to analyze and predict how these factors influence sales.  

🔹 **Key Findings:**  
✅ Higher temperatures = More ice cream sales 📈  
✅ Rainy days = Sales drop significantly 🌧️  
✅ Strong correlation exists between temperature and revenue  

---

## 📂 Dataset Information  
This dataset provides **temperature, revenue, and weather conditions** (rain/no rain) for ice cream sales. It helps explore trends and forecast future sales.  

📌 **Dataset Source:** [Download from Kaggle](https://www.kaggle.com/datasets/saliltirodkar/ice-cream-sales-analysis-temperature-and-weather?resource=download)  

---

## 📊 Project Workflow  

1️⃣ **Data Preprocessing** – Cleaning & preparing the dataset

3️⃣ **Applying Linear Regression** – Predicting sales
